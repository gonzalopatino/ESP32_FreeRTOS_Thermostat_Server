ThermoPatinoVerse

Smart IoT Thermostat Platform for the Modern Home

ESP32 Powered Cloud Connected Real-time Analytics

Project Overview

ThermoPatinoVerse is a complete IoT thermostat ecosystem combining embedded firmware, cloud infrastructure, and an intuitive web dashboard for monitoring and controlling your home climate with precision.

Built with a focus on reliability, security, and real-time performance, this platform demonstrates modern IoT best practices; from FreeRTOS task management on the edge device to secure API authentication and responsive data visualization in the cloud.

End-to-End Solution

From sensor to dashboard: a complete IoT architecture showcasing embedded systems, secure APIs, real-time data streaming, and modern web technologies.

System Architecture

ESP32 Firmware

FreeRTOS-based embedded application with modular task architecture for sensors, control logic, display, and network communication.

  • AHT20 Temperature Sensors
  • Hysteresis Control Algorithm
  • LCD Display Interface
  • Watchdog Task Monitoring
  • Secure HTTPS Telemetry

Cloud Backend

Django REST API with PostgreSQL database for device management, telemetry ingestion, and secure user authentication.

  • Django 5.x Framework
  • PostgreSQL Database
  • Device API Key Authentication
  • Rate Limiting & Security
  • RESTful API Design

Web Dashboard

Responsive web interface with real-time data visualization, dark/light themes, and intuitive device management.

  • Chart.js Visualizations
  • Bootstrap 5 UI Framework
  • Dark/Light Mode Toggle
  • Mobile-Responsive Design
  • CSV Data Export

Key Features

Precision Temperature Control

Configurable setpoint and hysteresis band for accurate climate management with support for heating and cooling modes.

Real-time Telemetry

Live streaming of temperature data with configurable sample rates and automatic dashboard updates every 15 seconds.

Secure Device Authentication

Per-device API keys with SHA-256 hashing, key rotation support, and timing-attack resistant validation.

Historical Analytics

Interactive temperature charts with custom date ranges, trend analysis, and CSV export for offline analysis.

Multi-Device Management

Register and monitor multiple thermostats from a single dashboard with per-device configuration and statistics.

Modern UI/UX

Clean, professional interface with dark/light theme support, smooth animations, and accessibility features.
